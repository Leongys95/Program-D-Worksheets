Learning Milestone Worksheet

Step 1

  1. Use class selector to select the div.container
  2. Save it as variable area
  3. Set area border to 5px outset
  4. Set area padding to 30px

Step 2

  1. Use element selector to select h1
  2. Save it as variable h1
  3. Set h1 text align to center
  4. Set h1 font size to 50px

Step 3

  1. Use query selector to select li
  2. Save it as variable li
  3. Set li text color to #09c

Step 4

  1. Use element selector to select ul
  2. Save it as variable ul
  3. Set ul text color to RGB values 153, 51, 51

Step 5

  1. Use query selector to select span inside h1
  2. Save it as variable span
  3. To change its background color later

Step 6

  1. Use three id selector to select three color buttons
  2. Save them as variables button1, button2, button3 respectively

Step 7

  1. Add event handler to all three color buttons
  2. When button click, change area background-color to the same color as button
  3. Set span background-color to the next color

Step 8

  1. Use chaining method to add event handler to three size buttons
  2. Without storing them to a variable
  3. When button click, change the area font size to respective size

Step 9

  1. Use querySelectorAll to select all li
  2. Save them as variable lis
  3. For each li, add a click event handler
  4. When li click, toggle the class "selected"
