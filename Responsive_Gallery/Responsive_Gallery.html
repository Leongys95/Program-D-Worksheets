Responsive Gallery Worksheet

Step 1

  1. Navigate to index.html
  2. Link index.html to Bootsrap using cdn
  3. Create a div.container
  4. Inside div.container, create a div.jumbotron
  5. Fill in h1 and p inside div.jumbotron

Step 2

  1. Create a select element
  2. Create four option elements in select
  3. For each option, fill in the value attribute

Step 3

  1. After div.jumbotron, create a new div
  2. Inside the div, create three img elements
  3. Leave the src empty
  4. Apply class img-fluid and img-thumbnail to each img

Step 4

  1. Proceed to script.js
  2. Link script.js to index.html
  3. Use querySelectorAll to selects all img elements
  4. Save them as variable images

Step 5

  1. Create an object named as imageGallery
  2. For each image category, create an object property
  3. For each object property, create an array to store three image names

Step 6

  1. Use chaining method to attach change event handler to select element
  2. When the content in the select element changed, do Step 7 and 8

Step 7

  1. If the value selected is not "none"
  2. Create a variable named pictures, retrieve the images of selected property from imageGallery
  3. For each image in variable pictures
  4. Set the corresponding image path to src attribute of HTML img element
  5. Set the CSS visibility to "visible"

Step 8

  1. Else if the value selected is "none"
  2. Remove src attribute from all img elements
  3. Set CSS visibility to "hidden"

Step 9

  1. Make the web page responsive to screen width
